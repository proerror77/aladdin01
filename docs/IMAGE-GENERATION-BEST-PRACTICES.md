## 模型选择策略

### 1. 定风格/生成主场景图

| 场景类型 | 推荐模型 | 备注 |
|---------|---------|------|
| 通用风格定义 | MJ (Midjourney) | 主推 |

| 场景类型 | 推荐模型 | 备注 |
|---------|---------|------|
| 通用 | 🍌Pro | 主推 |
| 东方面孔 | Seedream 4.5/4.6 | 注意图片的视角、景别一致 |

### 3. 分镜图/多角度图

| 优先级 | 模型 | 备注 |
|--------|------|------|
| 第一选择 | 🍌2 | 先尝试 |
| 备选 | 🍌Pro | 不达预期时使用 |

## 构图技巧

### 1. 前景遮挡

**目的**: 增加层次感和代入感，避免人物过于居中的平面感。

**示例**:
```
正面半身视角，红衣古装女侠手持深色酒壶饮酒，神情淡然凌厉，
伦勃朗光，强烈明暗对比与高反差，暗调氛围，高饱和低明度色调，
高曝微过曝，胶片颗粒感与肌理质感，
前景右下角酒壶遮挡，对焦模糊，
电影剧照质感，层次丰富，情绪张力拉满
```

**关键词**:
- 前景遮挡
- 对焦模糊
- 层次丰富

### 2. 引导线构图

**目的**: 将视线引向关键区域，增强画面深度和空间感。

**示例**:
```
大全景，覆雪的雪山峡谷入口，俯瞰视角，
女骑士身披深色斗篷、手持长剑立于崖边，
前方峡谷深渊被冰雪与云雾包裹，深不见底、向云层下无限延伸，
凸显极致深邃与冰雪凛冽感；
一只巨龙在她面前低空盘旋，
高山两侧矗立着极高的龙形建筑石像，威严感拉满，
两侧岩壁与龙形建筑的边缘，将视线引向峡谷深渊，
电影级写实画质
```

**关键词**:
- 引导线
- 视线引导
- 深邃感
- 无限延伸

### 3. 框架构图

**目的**: 利用门框、窗框等形成天然框架，增强画面的聚焦感和层次感。

**示例**:
```
真人实拍质感，女武将手持红缨长枪，站在沙漠土坯茶馆外，
落日黄昏时分，橙金与暗紫渐变天空，
茶馆木质门框与横梁形成天然框架，
门内可见模糊茶客身影，
暖黄侧逆光，框架内明暗对比，
尘土漫射光线，胶片颗粒感，浅景深
```

**关键词**:
- 框架构图
- 门框/窗框
- 框架内明暗对比
- 浅景深

### 4. 对称/中心构图

**目的**: 体现威严和庄重，适合宫廷、朝堂等场景。

**示例**:
```
超广角略俯拍，远景镜头，
使者跪拜在朝堂上，朝堂大殿挑高非常高和宏伟，
准备呈上奏折的姿态。
以大殿中轴线为基准，两边站着大唐洛阳时期的大臣。
使者正前方端坐着武皇武则天。
使者离武皇距离较远，武皇略微虚化，体现其威严的气势，
略微俯拍视角。宫殿非常庞大宏伟壮观
```

**关键词**:
- 对称构图
- 中轴线
- 威严感
- 略微虚化

## 减少 AI 味的关键词

### 1. 降低人物 AI 感

```
照片级写实，真人实拍质感
胶片颗粒，细微噪点
真实皮肤肌理，自然毛孔，肤质粗糙
人物的比例符合正常比例
```

**作用**:
- 强调真实视觉质感
- 模拟胶片拍摄的复古质感
- 避免过度磨皮的假感
- 确保人物符合物理世界规律

### 2. 镜头与摄影参数类

```
35毫米胶片拍摄
浅景深，对焦模糊，背景虚化
手持摄影，轻微运动模糊
低角度仰拍 / 鸟瞰俯拍
```

**作用**:
- 模拟真实电影机质感
- 虚化背景突出主体
- 增加真实临场感
- 让图片更有代入感

### 3. 光影与氛围类

```
伦勃朗光，强烈明暗对比
低饱和色调
镜头畸变，色差
高反差，暗调氛围
高饱和低明度色调
高曝微过曝
胶片颗粒感与肌理质感
```

**作用**:
- 明暗对比增强戏剧性
- 还原电影级沉稳色调
- 模拟真实光学镜头的物理缺陷
