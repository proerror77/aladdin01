# Seedance 2.0 提示词模板

## 核心公式

### 文生视频 (T2V)

```
[主体] + [动作] + [场景] + [镜头] + [风格] + [音频]
```

### 图生视频 (I2V)

```
主体+动作, 背景+动作, 镜头+运动
```

## 填空式模板

### 模板 1：人物动作场景

```
[角色描述]，[具体动作]，[场景环境]，[镜头类型]，[画面风格]
```

**示例**：
```
一位武术大师，在雨中练太极，慢动作，水滴飞溅，电影级光影
```

### 模板 2：自然景观

```
[自然元素]，[变化过程]，[环境描述]，[镜头运动]，[质感要求]
```

**示例**：
```
一朵玫瑰，从花苞到盛开的延时摄影，露珠滚落花瓣，背景虚化，4K微距
```

### 模板 3：镜头运动

```
镜头[运动方式]，[起始视角]到[结束视角]，[场景描述]，[时间/光线]，[风格]
```

**示例**：
```
镜头穿越东京涩谷十字路口的人群，从地面缓慢上升到鸟瞰视角，夜景霓虹灯，电影质感
```

### 模板 4：风格化场景

```
[风格标签]的[场景名称]，[特色元素1]，[特色元素2]，[氛围描述]
```

**示例**：
```
赛博朋克风格的长安城，飞行汽车穿梭在霓虹灯笼之间，古代宫殿与全息投影交融，雾气弥漫的街道
```

### 模板 5：超现实画面

```
[不可能的场景]，[奇幻元素]，[光影效果]，[拍摄方式]
```

**示例**：
```
一座中国古城从海底缓缓升起，鲸鱼在宫殿之间游过，阳光穿透海水照亮金色琉璃瓦，水下摄影
```

## 即梦官方格式（剧本模式）

```
【出镜角色-场景】
角色：<角色1>，<角色2>
场景：<场景名-时段（日/夜/黄昏/清晨）>
[画面]：[角色1]动作；[角色2]动作。[后景]：背景元素。

画面风格: 风格描述（画质/色调/质感）。
镜头1：[景别+运镜类型]，[角色]动作描述，开口说道："台词"（语气：情绪描述）
镜头2：[景别+运镜类型]，[角色]动作描述。画外音（角色名内心独白）："台词"（语气：情绪）

画面风格: 风格描述；搭配环境音效描述；禁止出现字幕、对话框、背景音乐
```

## 长镜次时间戳分镜法（13-15秒）

```
镜头1：全景固定镜头，0-3秒[角色]做X，4-8秒[角色]做Y，9-12秒[角色]做Z。
```

## 参考图引用（I2V）

```
角色：@图片1 作为<角色1>，@图片2 作为<角色2>
镜头描述中加：保持角色外观与参考图一致
```

## 长度限制

- 最大长度：2000 字符
- 推荐长度：500-1000 字符
- 超长提示词会被截断

## 禁止项（不要写进 prompt）

- ❌ 比例标注（`9:16` / `竖屏`）— 已是 API 参数
- ❌ 水印声明（`禁止出现水印`）— 已是 API 参数
- ❌ 画质标准（`4K Ultra HD`）— 已是 API 参数
