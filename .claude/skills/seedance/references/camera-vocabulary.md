# Seedance 2.0 镜头语言词汇库

## 景别 (Shot Size)

### 远景 (Extreme Long Shot / ELS)
- 展示大范围环境
- 人物在画面中很小
- 用于建立场景氛围

**示例**：
```
远景航拍，一座古城坐落在群山之中，云雾缭绕
```

### 全景 (Long Shot / LS)
- 展示人物全身和周围环境
- 强调人物与环境的关系

**示例**：
```
全景镜头，武术大师站在瀑布前，水雾弥漫
```

### 中景 (Medium Shot / MS)
- 拍摄人物膝盖以上
- 平衡人物和环境

**示例**：
```
中景，少女在樱花树下回眸，花瓣飘落
```

### 近景 (Close-Up / CU)
- 拍摄人物胸部以上
- 强调表情和情绪

**示例**：
```
近景特写，眼神中透着坚定，汗水滑落脸颊
```

### 特写 (Extreme Close-Up / ECU)
- 拍摄人物面部或物体细节
- 极致情感表达

**示例**：
```
极致特写，一滴眼泪从眼角滑落，睫毛颤动
```

## 运镜 (Camera Movement)

### 推镜头 (Push In / Dolly In)
- 镜头向前推进
- 聚焦主体，增强紧张感

**示例**：
```
镜头从远处缓缓推进，聚焦到女孩惊恐的眼神
```

### 拉镜头 (Pull Out / Dolly Out)
- 镜头向后拉远
- 揭示更大环境

**示例**：
```
镜头从特写拉远，露出整个空旷的大厅
```

### 摇镜头 (Pan)
- 镜头水平旋转
- 展示横向空间

**示例**：
```
镜头从左向右缓慢摇动，扫过整个城市天际线
```

### 跟镜头 (Tracking / Follow)
- 镜头跟随主体移动
- 保持主体在画面中心

**示例**：
```
镜头跟随少年在街道上奔跑，背景快速掠过
```

### 环绕镜头 (Orbit / Circle)
- 镜头围绕主体旋转
- 360度展示主体

**示例**：
```
镜头360度环绕武术大师，展示完整的战斗姿态
```

### 升降镜头 (Crane / Boom)
- 镜头垂直升降
- 改变视角高度

**示例**：
```
镜头从地面缓缓上升到鸟瞰视角，俯瞰整个广场
```

### 穿越镜头 (Through / Pass Through)
- 镜头穿过障碍物
- 创造空间连续性

**示例**：
```
镜头穿越人群，从拥挤的街道进入安静的咖啡馆
```

### 固定镜头 (Static / Locked)
- 镜头完全静止
- 强调画面稳定性

**示例**：
```
固定镜头，老人静坐在窗前，阳光洒在脸上
```

## 角度 (Camera Angle)

### 平视 (Eye Level)
- 与主体视线平齐
- 中性客观视角

**示例**：
```
平视角度，两人对视，眼神交流
```

### 俯视 (High Angle / Bird's Eye)
- 从上往下拍摄
- 显得主体渺小或脆弱

**示例**：
```
俯视角度，孤独的身影站在空旷的广场中央
```

### 仰视 (Low Angle / Worm's Eye)
- 从下往上拍摄
- 显得主体高大或威严

**示例**：
```
仰视角度，巨龙腾空而起，遮天蔽日
```

### 倾斜 (Dutch Angle / Canted)
- 镜头倾斜
- 营造不安或混乱感

**示例**：
```
倾斜镜头，走廊扭曲变形，营造诡异氛围
```

## 焦点 (Focus)

### 浅景深 (Shallow Depth of Field)
- 主体清晰，背景虚化
- 突出主体

**示例**：
```
浅景深，玫瑰花清晰，背景虚化成光斑
```

### 深景深 (Deep Depth of Field)
- 前景和背景都清晰
- 展示空间层次

**示例**：
```
深景深，从前景的花朵到远处的山脉都清晰可见
```

### 焦点转移 (Rack Focus)
- 焦点从一个主体转移到另一个
- 引导观众视线

**示例**：
```
焦点从前景的手表转移到背景的人物
```

## 节奏 (Pacing)

### 慢动作 (Slow Motion)
- 时间放慢
- 强调细节和情感

**示例**：
```
慢动作，水滴飞溅，每一滴都清晰可见
```

### 快动作 (Fast Motion / Time Lapse)
- 时间加速
- 展示变化过程

**示例**：
```
延时摄影，花朵从花苞到盛开，云层快速流动
```

### 定格 (Freeze Frame)
- 画面静止
- 强调关键瞬间

**示例**：
```
定格画面，足球即将入网的瞬间
```

## 转场 (Transition)

### 切换 (Cut)
- 直接切换
- 最常用的转场方式

**示例**：
```
镜头切换，从室内切到室外
```

### 淡入淡出 (Fade In / Fade Out)
- 画面渐显或渐隐
- 表示时间流逝

**示例**：
```
画面淡出，进入回忆场景
```

### 叠化 (Dissolve)
- 两个画面重叠过渡
- 柔和的转场效果

**示例**：
```
叠化转场，从春天的樱花叠化到秋天的枫叶
```

### 擦除 (Wipe)
- 一个画面擦除另一个
- 动感转场

**示例**：
```
擦除转场，新场景从左侧滑入
```

## 光影 (Lighting)

### 顺光 (Front Lighting)
- 光源在镜头后方
- 画面明亮，细节清晰

**示例**：
```
顺光拍摄，阳光照亮整个场景，色彩鲜艳
```

### 逆光 (Backlighting)
- 光源在主体后方
- 轮廓光，剪影效果

**示例**：
```
逆光剪影，人物站在夕阳前，轮廓清晰
```

### 侧光 (Side Lighting)
- 光源在主体侧面
- 强调立体感和质感

**示例**：
```
侧光照射，雕塑的纹理和阴影清晰可见
```

### 顶光 (Top Lighting)
- 光源在主体上方
- 戏剧化效果

**示例**：
```
顶光照射，舞台上的舞者，光束从上方洒下
```

### 底光 (Bottom Lighting)
- 光源在主体下方
- 诡异或超自然效果

**示例**：
```
底光照射，脸部阴影诡异，恐怖氛围
```

### 伦勃朗光 (Rembrandt Lighting)
- 侧光 + 反光板
- 经典肖像光

**示例**：
```
伦勃朗光，脸部一侧明亮，另一侧有三角形光斑
```

## 组合使用示例

### 示例 1：动作场景
```
近景跟镜头，武术大师在雨中出拳，慢动作，水滴飞溅，侧光照射，电影级光影
```

### 示例 2：情感场景
```
特写推镜头，眼泪从眼角滑落，浅景深，背景虚化，柔和顶光
```

### 示例 3：环境场景
```
远景航拍，镜头从云层中穿出，俯视整个城市，夕阳余晖，深景深
```

### 示例 4：转场场景
```
中景摇镜头，从左向右扫过整个房间，叠化转场到下一个场景
```
