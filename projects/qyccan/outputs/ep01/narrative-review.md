### shot-08

**问题**：`camera` 字段第二段为 `全景环绕镜头360度`，`seedance_prompt` 中为 `全景环绕镜头360度`。苏夜为拇指大小的蚕宝宝，对极小物体做 360 度精确环绕属于 Seedance 2.0 禁止运镜（微观环绕），模型无法执行。

```
```

```

**修复后**：
```
4-7秒：全景固定镜头（广角），家仆甲伸手抓向叶红衣，苏夜求生本能爆发，张嘴，小嘴化作漆黑漩涡黑洞迎风暴涨数米
```
```
4-7秒：全景固定镜头（广角），家仆甲伸手抓向[叶红衣]，[苏夜]求生本能爆发，张嘴，樱桃般的小嘴瞬间化作漆黑漩涡黑洞迎风暴涨数米...

**原因**：广角固定镜头可以在同一帧内展示苏夜（前景）、叶红衣（中景）、家仆甲（右侧）的三方关系，黑洞暴涨的视觉冲击不依赖环绕运镜，且 Seedance 2.0 对固定广角镜头的执行稳定性更高。
