# 开局青玉蚕 第9集

## 剧情摘要

天命之子登场，我才是反派？　　9-1：日，外，天风学院广场　　人物：导师，新生众，叶红衣，苏夜，萧凡（新角色/天命之子），青儿（身世神秘的绝美少女），戒指老爷爷（萧凡随身戒指里的灵魂，混响）　△空镜，天风学院广场人山人海　△广场中央梳理着新生入学测试资质的招牌，无数学生在这聚集。　　△一个高傲的导师指着一块巨大的测试石：把你们的手放上去，石头会根据血脉潜力发光。赤橙黄绿青蓝紫，紫色最高。△正围观的叶红衣十分兴奋，趴在叶红衣肩上的苏夜却嗅了嗅空气，随即表情不爽地看向一旁。叶红衣：好热闹啊。苏夜：（吸气声）怎么有股味？叶红衣：怎么了？苏夜：闻到了一股骄傲自大目中无人唯我独尊的臭味。　　△只见在苏夜和叶红衣前方，一个身穿朴素布衣，眼神坚毅的少年萧凡走上前，他身后跟着一只不起眼的小火鸟。他身边站着一位气质清冷、容貌绝世的少女青儿，正温柔地注视着他。苏夜（os）：简称，“主角味”。　　戒指老爷爷（混响）：小凡，稳住心神。沉寂三年，受尽白眼，是时候让这群有眼无珠的人看看你真正的天赋了！　　萧凡（嘴角勾起一抹自信的歪嘴笑，抚摸了一下手指上的古朴黑戒）：老师放心，曾经失去的，我今天全都要拿回来！　　△萧凡手掌按在测试石上，测试石爆发出璀璨的紫色光芒，直冲云霄！光芒中隐隐有一丝尊贵的金色火光！　　导师（激动）：紫...紫色潜力！百年难遇的天才！你叫什么名字？　　萧凡（不卑不亢，负手而立）：弟子萧凡。　　△全场轰动，无数少女对萧凡投去爱慕的目光。青儿更是骄傲地拉住他的衣角：“萧凡哥哥，青儿就知道你是最棒的。”赵无极则嫉妒地看着他。　　9-2：日，外，广场　　△轮到叶红衣。她把苏夜（缩小状态）放在石头上。导师（vo）：下一个，叶红衣！　　△苏夜懒洋洋地趴着，打了个哈欠。　　△测试石……突然剧烈震动，光芒非但没有亮起，反而散发出令人心悸的漆黑魔气！　　导师（大惊失色）：黑...黑气？这是大凶之兆！是反噬血脉！此女心术不正，她的灵兽更是邪魔之物！快拿下！　　众人惊恐后退：天啊，是邪魔！快杀了她！　　萧凡（皱眉，眼神闪过一丝厌恶）：青儿退后。此女身上的气息混沌无比，那黑蛇更是透着邪秽，天理难容。　　戒指老爷爷（混响）：小凡，小心那条蛇！它身上的气息……竟能吞噬天地气运，与你是宿命之敌！　　叶红衣（脸色苍白）：不可能！苏夜不是邪魔！　　苏夜（眯起眼）：一群白痴，居然敢说我和女人是邪魔？　　△苏夜突然张嘴，对着测试石咬了一口！咔嚓，黑气缭绕的测试石被啃出一个大洞。　　【系统】：检测到蕴含“天命气运”的测灵石，吞噬可掠夺气运之子的机缘。可以吃。　　△咔嚓！苏夜硬生生从测试石上咬下来一块！　　导师（眼珠子瞪出来）：他……他把测试反噬的魔石吃了？！　　△苏夜嚼了嚼，呸地吐出一口渣。　　苏夜：味道不错，就是有点硌牙。　　△突然，苏夜身上爆发出一股远超之前的恐怖气息，那股黑气被他完全吸收，并转化为精纯的力量，直接震碎了整块测试石！　　△轰隆！碎石飞溅。　　9-3：日，外，广场　　△烟尘中，苏夜淡定地游回叶红衣肩膀。　　苏夜：什么破石头，质量太差，居然炸了。这可怪不得我。　　导师（颤抖）：这……吞噬魔气化为己用？这究竟是什么怪物！　　△此时，学院深处传来一个苍老的声音（混响）：有趣，有趣。管他正道邪魔，力量才是根本。这女娃，特招入内核心院。　　全场哗然。　　萧凡（眉头紧锁，眼神冰冷，上前一步护住青儿，浑身散发着正道之光）（os）：此女与她的灵兽，气息诡异，绝非善类，日后必成大患。身为名门正派，我定要替天行道！　　苏夜（回头，对他咧嘴一笑）：看什么看，小子？再多管闲事，连你和你的小女友一起吃掉。
