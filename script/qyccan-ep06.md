# 开局青玉蚕 第6集

## 剧情摘要

家族大比，全员恶人　　6-1：日，外，擂台　　人物：叶家家主，长老们，叶红衣，叶如烟，各路龙套　　△擂台周围人山人海。　　△叶如烟脸上戴着面纱（遮疙瘩），站在擂台上，脚边趴着一只巨大的双头狼。　　叶如烟（得意）：下一位挑战者，叶红衣！姐姐，如果你现在跪下来求我，我也许会让我的“双头魔狼”轻一点咬。　　△台下哄笑。　　龙套C：叶红衣那条蚕子怕是已经被踩死吧？　　△叶红衣一身劲装，缓缓走上台。肩膀上空空如也。　　叶如烟：哟，你的灵兽呢？吓得不敢出来了？　　叶红衣：对付你，还不需要它出手。　　苏夜（声音从叶红衣袖子里传出，闷闷的）：放屁，明明是你嫌我刚才偷吃大蒜口臭，不让我出来！　　6-2：日，外，擂台　　人物：叶家家主，长老们，叶红衣，叶如烟，各路龙套　　△叶如烟大怒：给我撕碎她！　　△双头魔狼咆哮着冲向叶红衣。二阶巅峰妖兽的气势压得台下人喘不过气。　　△叶红衣拔剑，但双头狼速度太快，一爪拍飞了她的剑。　　叶红衣（后退）：可恶，境界差距太大了。　　叶如烟：去死吧！　　△双头狼张开血盆大口咬向叶红衣的喉咙。　　△千钧一发之际，一道绿光从叶红衣袖口射出！　　苏夜（VO）：吵死了！还让不让人睡觉了！　　△咻-嘭，苏夜的身形在空中迎风暴涨，鳞片发光！　　【系统】：技能“大小如意”发动！（虽然只能变大一点点）　　△原本筷子细的小蛇，瞬间变成了手臂粗细的蟒蛇！　　△苏夜一尾巴抽在双头狼的一个脑袋上！　　△音效：啪！！！　　△双头狼那个脑袋直接被抽歪了，庞大的身躯被抽得横飞出去，撞在擂台柱子上。　　6-3：日，外，擂台　　人物：苏夜（大蛇形态），全场众人　　△全场寂静。　　△苏夜盘在叶红衣面前，高昂着头颅，信子吞吐，发出的嘶嘶声竟然带着一股威压。　　苏夜（OS）：虽然我现在还是蛇，但我已经修出了“龙威”的一丝雏形。　　△那只双头狼爬起来，却夹着尾巴呜呜直叫，根本不敢看苏夜，趴在地上瑟瑟发抖。　　【系统】：检测到上位血脉压制。　　叶如烟（崩溃）：怎么可能！我的双头魔狼可是有啸月狼血统的！怎么会怕一条蛇！　　苏夜（回头看了叶红衣一眼）：愣着干嘛？那娘们没带灵兽，揍她啊！　　叶红衣心领神会，捡起剑，一步步走向叶如烟。叶红衣：现在，谁是废物？
